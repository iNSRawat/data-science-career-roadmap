# Free GitHub Repos to Practice Data Cleaning

> All messy. All free. Most tutorials give you clean data. Real jobs don't.

---

## 👶 For Complete Beginners

### 1. NamanGupta001 / Messy-Dataset-Cleaning

A single highly messy dataset covering missing values, wrong date formats, numeric columns full of symbols, and typographic errors. Good first project before anything more complex.

- **GitHub:** https://github.com/NamanGupta001/Messy-Dataset-Cleaning

---

## 🐍 Python Data Cleaning

### 2. sumony2j / Data_Cleaning_Preprocessing

Cleaning and preprocessing across multiple datasets using Python libraries. Covers the full range of issues you will actually encounter in a real role.

- **GitHub:** https://github.com/sumony2j/Data_Cleaning_Preprocessing

### 3. pb111 / Data-Cleaning-with-Python-and-Pandas

Comprehensive walkthrough of cleaning techniques using NumPy and Pandas. Covers missing values, outliers, duplicates, and untidy data structures.

- **GitHub:** https://github.com/pb111/Data-Cleaning-with-Python-and-Pandas

---

## 🧰 Multi-Tool Projects (Python, SQL, Excel, Power BI)

### 4. eyowhite / Messy-dataset

Messy HR and healthcare datasets cleaned across Python, Excel, SQL, and Power BI. Good for seeing how the same problem is solved across different tools.

- **GitHub:** https://github.com/eyowhite/Messy-dataset

### 5. ojalp26 / Cleaning-dirty-data-samples

Eight dirty data samples cleaned using Power Query in Excel. Sales data, customer records, hospital data, invoices. Covers the kinds of structural mess you will actually find in real files.

- **GitHub:** https://github.com/ojalp26/Cleaning-dirty-data-samples

---

## ⚔️ Dirty Data Challenges

### 6. OxfordIHTM / messy-data

Real datasets from Ministry of Health records including vaccine studies and population data. Harder than most practice sets because it comes from actual government systems.

- **GitHub:** https://github.com/OxfordIHTM/messy-data

---

## 💪 Advanced Cleaning (Text, Dates, and More)

### 7. misszeferino / Nashville-Housing-Data-Cleaning

The Nashville Housing dataset cleaned in SQL. Covers date standardisation, address parsing, NULL handling, and duplicate removal. A well-known portfolio project for good reason.

- **GitHub:** https://github.com/misszeferino/Nashville-Housing-Data-Cleaning

### 8. underthecurve / pandas-data-cleaning-tricks

Real-world messy datasets with documented Python cleaning tricks. Good for when you know the basics and want to handle the edge cases.

- **GitHub:** https://github.com/underthecurve/pandas-data-cleaning-tricks
