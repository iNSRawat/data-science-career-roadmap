## 🎯 Overview

This repository provides a **complete, structured roadmap** for career changers (especially from digital marketing) to break into data science. Built on real-world industry insights from 2024-2025, this guide includes:

✅ **12-month learning path** with weekly milestones  
✅ **100% free resources** (courses, tools, datasets)  
✅ **5+ portfolio project templates** with business impact  
✅ **Interview prep materials** (SQL, Python, ML, behavioral)  
✅ **Resume & LinkedIn optimization** for career changers  
✅ **Salary data** across US, India, and global markets  

**Target Audience**: Career changers from non-technical backgrounds, particularly marketing professionals leveraging their domain expertise.

## 📊 Career Outlook

### Role Definition

**Data Scientists** extract actionable insights from complex datasets using statistics, machine learning, and domain expertise to solve business problems and drive strategic decisions.

### Daily Responsibilities

- Collecting and cleaning data from multiple sources
- Exploratory data analysis (EDA) to identify patterns
- Building predictive models using machine learning
- Creating dashboards and visualizations for stakeholders
- Deploying models into production systems
- Presenting findings to non-technical business leaders

### Career Progression

| Level | Title | Experience | Salary (US) | Salary (India) |
|-------|-------|------------|-------------|----------------|
| Entry | Junior Data Scientist | 0-2 years | $85K-$120K | ₹6-12 lakhs |
| Mid | Data Scientist | 3-5 years | $130K-$170K | ₹15-30 lakhs |
| Senior | Senior Data Scientist | 5-7 years | $160K-$210K | ₹25-45 lakhs |

## 🎨 Portfolio Projects

### Project Template Structure

```
project-name/
├── README.md (Problem, approach, results, impact)
├── data/ (sample data or link)
├── notebooks/ (Jupyter notebooks with analysis)
├── src/ (Python scripts, modular code)
├── models/ (trained models)
├── visualizations/ (key charts)
├── app/ (deployment files if applicable)
└── requirements.txt (dependencies)
```

### 5 Must-Have Portfolio Projects

#### 1. **Customer Churn Prediction** (Classification)
**Business Problem**: Predict which customers are likely to cancel subscriptions

**Skills Demonstrated**:
- Data cleaning & EDA
- Feature engineering
- Multiple ML models (Logistic, RF, XGBoost)
- Model evaluation & comparison
- Business recommendations

**Datasets**: Telecom, SaaS, or subscription service churn data (Kaggle)

---

#### 2. **Marketing Campaign ROI Optimizer** (Regression + Time Series)
**Business Problem**: Forecast campaign performance and optimize budget allocation

**Skills Demonstrated**:
- Time series analysis
- Prophet forecasting
- SQL data extraction
- Interactive dashboard (Tableau/Plotly)
- A/B testing analysis

**Your Edge**: Leverage your marketing background!

---

#### 3. **Customer Review Sentiment Analysis** (NLP)
**Business Problem**: Analyze customer feedback to identify product issues

**Skills Demonstrated**:
- Text preprocessing
- TF-IDF vectorization
- Sentiment classification
- Topic modeling
- Visualization of insights

**Datasets**: Amazon reviews, Yelp reviews, app store reviews

---

#### 4. **Sales Forecasting Dashboard** (Time Series + BI)
**Business Problem**: Predict next quarter sales for inventory planning

**Skills Demonstrated**:
- ARIMA/Prophet models
- Seasonal decomposition
- Interactive dashboard
- Business intelligence
- Actionable insights

**Tools**: Python + Tableau Public or Plotly Dash

---

#### 5. **Recommendation System** (Collaborative Filtering)
**Business Problem**: Recommend products/content to users

**Skills Demonstrated**:
- Collaborative filtering
- Matrix factorization
- Evaluation metrics (RMSE, precision@K)
- Scalability considerations
- A/B testing framework

**Datasets**: MovieLens, Amazon products, Last.fm music

**Job Boards**:
- LinkedIn Jobs (Entry Level filter)
- Indeed
- Glassdoor

- Built In (tech companies)
- Kaggle Jobs

**Company Types to Target**:
1. **Startups (10-100 employees)** - More flexible on requirements
2. **Marketing/AdTech companies** - Value domain expertise
3. **E-commerce** - Appreciate growth mindset
4. **Consulting firms** - Entry-level programs
5. **Mid-size tech** - Less competitive than FAANG

**Application Strategy**:
- Apply to **20-30 jobs/week**
- Tailor resume for each (15-20 min)
- Follow up with hiring managers on LinkedIn
- Track applications in spreadsheet

### Interview Preparation

**Technical Topics**:
- **SQL**: 50+ HackerRank/StrataScratch problems
- **Python**: LeetCode Easy (30-50 problems)
- **Statistics**: Hypothesis testing, distributions, p-values
- **ML**: Explain algorithms, when to use, evaluation metrics
- **Case Studies**: Structure data science problems

**Behavioral (STAR Method)**:
Prepare 5-7 stories about:
- Problem-solving
- Collaboration
- Leadership
- Learning agility
- Handling failure

**Resources**:
- 📖 [Cracking the Data Science Interview](https://www.amazon.com/Cracking-Data-Science-Interview/dp/B0B5XVY5R6)
- 💻 [InterviewQuery](https://www.interviewquery.com/)
